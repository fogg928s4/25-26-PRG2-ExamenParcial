# Entrega Examen Parcial - PRG2 🐌

Este directorio incluye la entrega del examen parcial de la materia **Programación 2** a nombre de José Melgares.

## Estructura

Dentro de la carpeta `src` se encuentra las soluciones propuestas a las distintas vistas públicas y privadas de la solución. Del mismo modo, dentro de `src/assets` se encuentran la clase auxiliar de `Consola`.

La implementación de las vistas privadas de las clases `Película`, `Sala`, `Cine`, `Proyeccion`, y `FechaHora` se encuentran dentro de `src`.
