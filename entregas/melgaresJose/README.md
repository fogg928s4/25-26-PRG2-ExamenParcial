# Entrega Examen Parcial - PRG2 🐌

Este directorio incluye la entrega del examen parcial de la materia **Programación 2** a nombre de José Melgares.

## Estructura

Dentro de la carpeta `src` se encuentra las soluciones propuestas a las distintas vistas públicas y privadas de la solución. Del mismo modo, dentro de `src/classes` se encuentran la clase auxiliar de `Consola`.

La vista pública de la parte 2 se encuentra en la clase `Cliente` dentro de `src`. Por otro lado,la vista privada se encuentra dentro del directorio anteriormente mencionado: `src/classes`.
